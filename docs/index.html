翻四張牌，用整數加減乘除湊出目標數字！

⚠️ GitHub Pages 只有遊戲畫面，還需要 Render 後端。
請先到 render.com 部署後端，再把網址填進 docs/config.js 的 SERVER_URL。
或者直接用 Render 給你的網址玩（更簡單）。

怎麼跟朋友玩？

  1. 把遊戲網址傳給朋友（兩人開同一個連結）
  2. 玩家 A 按「建立房間」，得到 4 位房間代碼
  3. 把房間代碼傳給玩家 B（LINE、Discord 都行）
  4. 玩家 B 輸入代碼按「加入房間」
  5. 畫面顯示「兩位玩家已連線」就可以開始！
